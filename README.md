# 👥 KeenKeeper

**Live Demo:** https://keenkeeper-sh.vercel.app/

---

## 📖 About the Project

KeenKeeper is a simple yet powerful web application designed to help you
maintain meaningful friendships. In our busy lives, it's easy to lose touch with
people we care about. This app lets you keep track of your friends, log your
interactions, and see at a glance who needs some love and attention.

Whether it's a quick text, a phone call, or a video chat, KeenKeeper helps you
remember to stay connected with the people that matter most.

---

## 🎯 Key Features

- **Friend Management** — Keep a personal list of your friends with their
  photos, contact info, and relationship goals
- **Interaction Tracking** — Log calls, texts, and video chats with a single
  click and watch your timeline grow
- **Smart Analytics** — Visual charts show you exactly which types of
  interactions you're doing most (calls vs texts vs videos)
- **Responsive Design** — Works perfectly on mobile, tablet, and desktop—stay
  connected from anywhere

---

## 🛠️ Technologies Used

- **Next.js 15** — Fast, modern React framework for production
- **Tailwind CSS** — Beautiful, utility-first styling
- **DaisyUI** — Pre-built components for faster development
- **Recharts** — Interactive charts and data visualization
- **React Icons** — Clean, consistent icons throughout the app
- **Sonner** — Toast notifications for user feedback

---

## 💡 How It Works

1. **Browse Your Friends** — See all your friends on the home page with their
   current status (On Track, Almost Due, Overdue)
2. **View Details** — Click on a friend to see their profile and interaction
   history
3. **Check In** — Log a call, text, or video chat from the friend's detail page
4. **Track Over Time** — Visit the Timeline page to see all your interactions in
   chronological order
5. **Analyze Your Patterns** — Head to the Stats page to see a pie chart of your
   interaction types

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/keenkeeper_sh.git

# Navigate to the project
cd keenkeeper_sh

# Install dependencies
npm install

# Start the development server
npm run dev
```
